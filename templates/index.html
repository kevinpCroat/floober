Here's the fancy floober api docs

Request: Total trips driven (endpoints don't end in slashes)
URI: /api/trips

Request: Total trips driven in date range YYYYMMDD => all endpoints use this fmt
URI: /api/trips/:start_date/:end_date

Request: Total number of clients w/ optional date range
URI: /api/clients/:start_date/:end_date

Request: Total trips driven in the last hr
URI: /api/trips/hour

Request: Total miles per client w/ optional date range
URI: /api/clients/miles/:start_date/:end_date

Request: Avg for a city given center point and radius w/optional date range
URI: /api/trips/fare/avg/city/:radius/:lat/:lon/:start_date/:end_date

Request: Median rating for a driver w/ optional date range
URI: /api/driver/:driver_id/rating/median/:start_date/:end_date
